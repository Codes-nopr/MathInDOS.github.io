Console applications

Console means it have no graphical interface and console applications are used from a application. In Windows it's cmd or Command Prompt and in Linux it's Terminal (Some linux distributions use other names like sakura or xTerminal etc.) It's used to open console applications, batch configration and some other tasks. But to execute programs you also need to provide arguments in cmd.

How I give arguments?

To give arguments of application just locate the directory where the app stay. To locate just type cd/pushd "directory" and type app name with '-h or --h or /?' to get help about the parameters.
